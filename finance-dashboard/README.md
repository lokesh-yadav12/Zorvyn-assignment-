# Finance Dashboard

A modern finance tracking dashboard built with React, TypeScript, and Tailwind CSS.

## Features

- **Dashboard Overview**: View your financial summary with interactive charts
- **Transaction Management**: Track income and expenses with detailed categorization
- **Financial Insights**: Get smart recommendations based on your spending patterns
- **Role-Based Access**: Switch between Admin and Viewer modes
- **Dark Mode**: Toggle between light and dark themes
- **Export Data**: Download your transactions as CSV

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Recharts
- Vite

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Usage

The app starts with mock data to demonstrate functionality. You can:
- Add, edit, or delete transactions (Admin mode)
- Filter and search transactions
- View spending insights and trends
- Export data to CSV
- Switch between light and dark themes
